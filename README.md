## 📋 Descrição

Primeiro projeto do meu portfólio de análise de dados, desenvolvido durante o curso **Santander + DIO**. O objetivo é consolidar os fundamentos de Python, manipulação de arquivos CSV e versionamento com Git/GitHub.

O script analisa um arquivo de vendas e gera um relatório completo com métricas de negócio.

## 🎯 Funcionalidades

- ✅ Leitura de arquivo CSV com dados de vendas
- ✅ Cálculo do faturamento total
- ✅ Identificação do produto mais vendido
- ✅ Média de vendas por dia
- ✅ Top 3 dias com maior faturamento
- ✅ Resumo por produto (quantidade e faturamento)
- ✅ Geração de relatório em .txt
- ✅ Tratamento de erros

## 🚀 Como Executar

### Pré-requisitos
- Python 3.8 ou superior instalado
- Git instalado (opcional, apenas para clonar)

### Passo a Passo

1. **Clone o repositório**
   ```bash
   git clone https://github.com/mayconaap/analise-vendas-python.git

2. **Entre na pasta do projeto**
   ```bash
   cd analise-vendas-python

3. **Execute o script Python**
   ```bash
    python analise_vendas.py

4. **Quando o programa perguntar:**
   ```bash
   💾 Deseja salvar este relatório em um arquivo? (s/n):
   Digite s (a letra "s" minúscula) e pressione Enter

5. **Pronto! O relatório será salvo como relatorio_vendas.txt na mesma pasta**
